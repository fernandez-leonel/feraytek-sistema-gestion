# Feraytek – Sistema de Gestión

## Descripción
Feraytek es un sistema de gestión desarrollado como proyecto académico, orientado a la administración de información mediante una arquitectura cliente-servidor.

El sistema está compuesto por una API central, una aplicación web y una aplicación de escritorio, permitiendo una gestión integrada de los datos.

## Tecnologías utilizadas
- Backend: .NET / Node.js
- Base de datos: SQL
- Web: HTML, CSS, JavaScript
- Control de versiones: Git y GitHub

## Trabajo en equipo y rol desempeñado
Proyecto desarrollado de forma **grupal**.

Si bien el trabajo fue realizado en equipo, mi participación estuvo enfocada principalmente en:
- Desarrollo de la **API** y lógica backend
- Diseño y uso de la **base de datos SQL**
- Desarrollo de la **aplicación web**
- Desarrollo de la **aplicación de escritorio**
- Integración entre los distintos componentes del sistema
- Organización de tareas y coordinación del trabajo en distintas etapas del proyecto
